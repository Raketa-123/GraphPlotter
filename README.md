# GraphPlotter

Веб-приложение на базе Django для интерактивного построения, визуализации и хранения математических функций. Проект предоставляет пользователям личный кабинет с возможностью генерации графиков, ведения истории запросов и поиска по ранее созданным визуализациям.

## Функциональные возможности

* **Аутентификация и авторизация:** Безопасная регистрация и вход пользователей сессионным методом для веб-интерфейса, а также поддержка JWT (JSON Web Tokens) для интеграции с API.
* **Визуализация функций:** Автоматический парсинг математических выражений (включая линейные, степенные и тригонометрические функции) и генерация графиков в формате PNG без использования графической оболочки сервера (мод `Agg` в Matplotlib).
* **История и поиск:** Автоматическое сохранение сгенерированных графиков в профиле пользователя, вывод истории запросов в хронологическом порядке и фильтрация по названию функции.
* **Изоляция данных:** Каждый пользователь имеет доступ исключительно к своей истории построенных графиков.

## Технологический стек

* **Бэкенд:** Python, Django, Django REST Framework (DRF), SimpleJWT
* **Графика и математика:** Matplotlib, NumPy
* **Фронтенд:** HTML5, CSS3 (адаптивная верстка, боковая панель истории), JavaScript

## Архитектура проекта

Проект разделен на две основные составляющие:
1.  **Monolith Web (View):** Традиционные представления Django, отвечающие за рендеринг HTML-страниц (`RegisterPage`, `LoginPage`, `HomePage`, `LogoutPage`).
2.  **REST API (APIView):** Эндпоинты для программного взаимодействия и авторизации через токены (`RegisterView`, `LoginView`).

Генерация изображений происходит в оперативной памяти сервера через буфер `io.BytesIO` с последующим сохранением в файловую систему в виде объектов `ContentFile`.

## Инструкция по установке и запуску

1. Клонируйте репозиторий
```
  git clone https://github.com/Raketa-123/GrapsPlotter.git
  cd GrapsPlotter
```

2. Создайте и активируйте виртуальное окружение:
```
   python -m venv venv
```
 # Для Windows:
```
   venv\Scripts\activate
```
  # Для macOS/Linux:
```
   source venv/bin/activate
```

3. Установите необходимые зависимости:
```
   pip install -r requirements.txt
```

4. Выполните миграции базы данных:
```
   python manage.py makemigrations
   python manage.py migrate
```
5. Запустите локальный сервер разработки:
```
   python manage.py runserver
```
